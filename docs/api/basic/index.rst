Basic Module
============

Module for basic preprocessing techniques: lowercase, urls replacement
symbols replacement, multipart words union, etc.

Some extra techniques very useful the testing of `preprocess` library in
real scenarios are included in this module :func: `replace_point_secuence`,
:func: `add_doc_end_point`.
